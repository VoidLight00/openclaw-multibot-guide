3. AI 모델 설정

Claude, Gemini, GPT 등 다양한 AI 모델을 연동합니다. OAuth와 API Key 방식의 차이점과 설정 방법을 알아봅니다.

[MODEL] 지원 모델

제공자 모델 인증 방식 비용
Anthropic Claude Opus 4.5, Sonnet 4.5 OAuth / API Key 유료
Google Gemini 2.0 Flash, Pro OAuth / API Key 무료 티어 있음
OpenAI GPT-4o, GPT-4 Turbo API Key 유료
OpenRouter 다양한 모델 API Key 모델별 상이

[KEY] 인증 방식

OAuth (권장)

RECOMMENDED

  • 브라우저로 로그인
  • API 키 관리 불필요
  • 자동 토큰 갱신

지원: Anthropic, Google

API Key

TRADITIONAL

  • 직접 키 발급 필요
  • 환경변수 또는 config에 저장
  • 키 관리 필요

지원: 모든 제공자

[SETUP] openclaw configure 완전 가이드

openclaw configure는 AI 모델과 채널을 설정하는 대화형 마법사입니다.

Terminal 
openclaw configure

Step 1: Gateway 위치 선택

프롬프트 
"Where will the Gateway run?"
→ Local (this machine)  [Enter]

Step 2: 섹션 선택

화살표 키(↓)와 스페이스바로 선택:

  1. Model 이동 → 스페이스바 (체크)
  2. Channels 이동 → 스페이스바 (체크)
  3. Continue 이동 → Enter

Step 3: Model Provider 선택

사용할 AI 모델 제공자를 선택합니다.

[CLAUDE] Anthropic (Claude) OAuth 설정

Anthropic 선택

프롬프트 
"Select model providers"
→ Anthropic (Claude)  [스페이스바로 선택]
→ Continue  [Enter]

인증 방식 선택

프롬프트 
"Anthropic authentication"
→ OAuth (browser login)  [권장]
   또는
→ API Key (manual entry)

OAuth 선택 시: 브라우저 인증

자동으로 브라우저가 열립니다.

  1. Anthropic 계정으로 로그인
  2. "OpenClaw" 앱 권한 승인
  3. 터미널로 돌아오면 완료

API Key 선택 시: 키 입력

프롬프트 
"Enter Anthropic API Key"
→ sk-ant-api03-xxxxxxxxxxxxx  [붙여넣기]

API 키: console.anthropic.com

[!] Claude 구독 사용 경고

Claude Pro/Max 구독 계정을 OAuth로 연동하면 계정 정지 위험이 있습니다. 자동화 툴 사용이 이용약관에 위반될 수 있습니다. API Key 방식 권장.

[GOOGLE] Google Provider 종류

Google Gemini를 사용하는 방법은 두 가지가 있습니다. 멀티봇 운영 시 반드시 확인하세요!

Provider API 백엔드 쿼터 추천 용도
google-gemini-cli Google Code Assist (일반 Gemini) 제한적 (소비자용) 단일 봇, 가벼운 사용
google-antigravity Cloud Code Assist 관대함 (개발자 도구용) 멀티봇, 고빈도 사용
[!] 멀티봇 운영 시 주의

여러 봇이 google-gemini-cli를 사용하면 쿼터를 빠르게 소진합니다 (429 에러). google-antigravity provider를 사용하세요. Google One 구독과는 별개입니다.

Antigravity Provider 설정

Terminal 
# 1. 플러그인 활성화
openclaw plugins enable google-antigravity-auth

# 2. OAuth 로그인
openclaw models auth login --provider google-antigravity

# 3. 에이전트 모델 설정 (openclaw.json)
{
  "model": {
    "primary": "google-antigravity/gemini-3-pro-high"
  }
}

사용 가능한 Antigravity 모델

Terminal 
# 모델 목록 확인
openclaw models list --all | grep antigravity
모델 설명
google-antigravity/gemini-3-pro-high Gemini 3 Pro (고품질)
google-antigravity/gemini-3-pro-low Gemini 3 Pro (저지연)
google-antigravity/gemini-3-flash Gemini 3 Flash (빠름)

[GOOGLE-CLI] Google Gemini CLI OAuth 설정

단일 봇이나 가벼운 사용에는 기본 google-gemini-cli도 사용 가능합니다.

Google 선택

프롬프트 
"Select model providers"
→ Google (Gemini)  [스페이스바로 선택]
→ Continue  [Enter]

인증 방식 선택

프롬프트 
"Google authentication"
→ OAuth (browser login)  [권장]
   또는
→ API Key (Google AI Studio)

OAuth 선택 시: 브라우저 인증

  1. Google 계정 선택
  2. "OpenClaw" 앱 권한 승인
  3. "계속" 클릭
  4. 터미널로 돌아오면 완료

API Key 선택 시: 키 입력

프롬프트 
"Enter Google AI API Key"
→ AIzaSyxxxxxxxxxxxxxxxxx  [붙여넣기]

무료 API 키: aistudio.google.com/apikey

[i] Gemini 무료 티어

Google AI Studio의 Gemini Flash는 분당 요청 제한이 있지만 무료로 사용 가능합니다.

[GPT] OpenAI (GPT) 설정

OpenAI는 API Key 방식만 지원합니다. (OAuth 없음)

OpenAI 선택

프롬프트 
"Select model providers"
→ OpenAI (GPT)  [스페이스바로 선택]
→ Continue  [Enter]

API Key 입력

프롬프트 
"Enter OpenAI API Key"
→ sk-proj-xxxxxxxxxxxxx  [붙여넣기]

API 키: platform.openai.com/api-keys

[MULTI] 여러 Provider 동시 설정

한 번에 여러 Provider를 선택할 수 있습니다.

예시: Claude + Gemini 동시 설정 
"Select model providers"
→ [x] Anthropic (Claude)   [스페이스바]
→ [x] Google (Gemini)      [스페이스바]  
→ [ ] OpenAI (GPT)
→ Continue  [Enter]

# 이후 각 Provider별로 인증 진행

[AFTER] 설정 완료 후

Terminal 
# Gateway 재시작
openclaw gateway restart

# 상태 확인
openclaw status

# 설정된 모델 확인
openclaw models list

방법 2: 웹 UI Dashboard

Dashboard 열기

Terminal 
openclaw dashboard

브라우저가 자동으로 열립니다. 토큰이 포함된 URL:

http://localhost:18789/?token=xxxxx

Auth 섹션 이동

좌측 메뉴 또는 Settings에서 Authentication 찾기

Google 프로필 추가

Auth Profiles → Add Entry → google:default 생성

  • Provider: Google
  • Mode: oauth

Connect 클릭

Google 로그인 창이 열리면 승인

방법 3: CLI로 직접 설정

config 값을 직접 설정하는 방법:

Terminal 
# Google OAuth 프로필 추가
openclaw config set auth.profiles.google:default.provider google
openclaw config set auth.profiles.google:default.mode oauth

# Gateway 재시작
openclaw gateway restart

재시작 후 Dashboard에서 Connect 버튼으로 실제 인증을 완료합니다.

[FIX] 문제 발생 시: Doctor 사용

설정이 꼬였을 때 doctor 명령어로 진단:

Terminal 
# 헬스 체크 및 자동 수정
openclaw doctor

# 상태 확인
openclaw status

# 버전 확인 (업데이트 필요할 수 있음)
openclaw --version
[i] 버전 업데이트

오래된 버전에서는 일부 명령어가 없을 수 있습니다. 최신 버전으로 업데이트하세요:

npm update -g openclaw

[ANTHROPIC] Claude 설정

OAuth 방식

Google과 동일하게 openclaw configure에서 Anthropic 선택

API Key 방식

~/.openclaw/.env 
ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxxxxxxxxxx

API 키는 Anthropic Console에서 발급

[OPENAI] GPT 설정

OpenAI는 API Key 방식만 지원합니다.

~/.openclaw/.env 
OPENAI_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxx

API 키: OpenAI Platform

[CONFIG] 에이전트별 모델 지정

openclaw.json 
{
  "agents": {
    "list": [
      {
        "id": "agent-claude",
        "workspace": "/path/to/workspace1",
        "model": {
          "primary": "anthropic/claude-opus-4-5"
        }
      },
      {
        "id": "agent-gemini",
        "workspace": "/path/to/workspace2",
        "model": {
          "primary": "google/gemini-2.0-flash"
        }
      }
    ]
  }
}

[RUNTIME] 런타임 모델 변경

채팅 중 모델 변경:

텔레그램에서 
/model opus      # Claude Opus로 변경
/model sonnet    # Claude Sonnet으로 변경
/model flash     # Gemini Flash로 변경

[COMPARE] 모델 비교

모델 장점 추천 용도
Claude Opus 최고 성능, 복잡한 추론 분석, 중요 결정
Claude Sonnet 균형 잡힌 성능/비용 일반 작업, 코딩
Gemini Flash 빠름, 저렴/무료 간단한 대화, 리마인더
GPT-4o 멀티모달, 빠름 이미지 분석, 범용
