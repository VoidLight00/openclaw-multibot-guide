활용 예시

실제로 멀티봇을 어떻게 활용할 수 있을까요? 다양한 구성 예시와 사용 시나리오를 소개합니다.

[MASK] 예시 1: 악마 삼총사

각기 다른 성격의 악마 컨셉 에이전트 3명

벨페고르

Claude Opus

  • 역할: 트레이딩, 복잡한 분석
  • 성격: 나태하지만 일할 땐 제대로
  • 말투: 직설적, 반말

[MONEY] 마몬

Gemini Flash

  • 역할: 리마인더, 일정 관리
  • 성격: 효율 집착, 잔소리
  • 말투: 친절하지만 간결

아스모데우스

Gemini Flash

  • 역할: 코딩, 개발
  • 성격: 꼼꼼, 기술 오타쿠
  • 말투: 전문적, 상세 설명

[WORK] 예시 2: 업무 분리형

에이전트 담당 모델 사용 시간
Work Bot 업무 관련 질문 Claude Sonnet 평일 9-18시
Life Bot 일상, 취미 Gemini Flash 항상
Code Bot 개발 전용 Claude Sonnet 필요 시

[i] 예시 3: 모델 티어링

비용 최적화를 위한 모델 분리

구성

  • Premium Bot (GPT-4 / Claude Opus) - 중요한 분석, 복잡한 작업
  • Standard Bot (Claude Sonnet) - 일반 대화, 작업
  • Quick Bot (Gemini Flash) - 간단한 질문, 빠른 응답
Config 예시 
{
  "agents": {
    "list": [
      {
        "id": "premium",
        "workspace": "~/.openclaw/workspace-premium",
        "model": { "primary": "anthropic/claude-opus-4-5" }
      },
      {
        "id": "standard",
        "workspace": "~/.openclaw/workspace-standard",
        "model": { "primary": "anthropic/claude-sonnet-4-5" }
      },
      {
        "id": "quick",
        "workspace": "~/.openclaw/workspace-quick",
        "model": { "primary": "google/gemini-2.0-flash" }
      }
    ]
  }
}

[HOME] 예시 4: 가족 공유

가족 구성원별 전용 봇

[i] 활용법

각 가족 구성원이 자신만의 봇과 대화. USER.md에 각 사람의 정보를 설정하면 개인화된 응답이 가능합니다.

봇 사용자 특화 기능
@dad_assistant_bot 아빠 뉴스, 스포츠, 업무
@mom_assistant_bot 엄마 요리, 쇼핑, 일정
@kid_assistant_bot 아이 숙제 도움, 퀴즈 (필터링 강화)

[QUICK] 고급 팁

에이전트 간 통신

에이전트끼리 메시지를 주고받을 수 있습니다.

SOUL.md에서 설정 
## 다른 에이전트 호출
복잡한 코딩 작업은 아스모데우스에게 위임할 수 있다.
sessions_send로 작업을 전달하고 결과를 받아온다.

공유 메모리

옵시디언 같은 공유 볼트를 여러 에이전트가 접근:

각 에이전트의 TOOLS.md 
## 공유 볼트
경로: /Users/username/Obsidian/SharedVault/

이 볼트는 모든 에이전트가 접근 가능.
- 벨페고르: 트레이딩 일지
- 마몬: 할 일 목록
- 아스모데우스: 개발 노트

시간대별 모델 전환

HEARTBEAT.md와 cron을 활용해 시간대별로 모델을 전환할 수 있습니다.

아이디어 
# HEARTBEAT.md
## 모델 전환 규칙
- 업무 시간 (9-18시): Opus 사용
- 그 외 시간: Sonnet 사용
- 심야 (0-6시): Flash만 사용 (비용 절감)

[CHART] 비용 추정

3개 에이전트 운영 시 예상 비용 (활발한 사용 기준)

모델 일 사용량 월 예상 비용
Claude Opus ~50K tokens ~$30-50
Claude Sonnet ~100K tokens ~$10-20
Gemini Flash ~200K tokens 무료 (제한 내)
[i] 비용 절감 팁

간단한 작업은 Gemini Flash에 맡기고, 복잡한 작업만 Opus/Sonnet을 사용하면 비용을 크게 줄일 수 있습니다.
