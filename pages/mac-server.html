Mac Mini 서버 설정

Mac Mini를 24/7 OpenClaw 서버로 운영하기 위한 필수 설정. 잠자기 비활성화, 자동 재시작, Gateway 자동 실행까지.

[1] 잠자기 완전 비활성화

Mac Mini가 서버로 동작하려면 절대 잠들면 안 됩니다.

Terminal (한 줄로 실행) 
sudo pmset -a sleep 0 disksleep 0 displaysleep 0 hibernatemode 0 standby 0 autopoweroff 0 autorestart 1 womp 1

설정 상세

옵션 값 설명
sleep 0 끔 시스템 잠자기 비활성화
disksleep 0 끔 디스크 잠자기 비활성화
displaysleep 0 끔 화면 잠자기 비활성화
hibernatemode 0 끔 최대 절전 모드 비활성화
standby 0 끔 대기 모드 비활성화
autopoweroff 0 끔 자동 전원 끄기 비활성화
autorestart 1 켬 정전 후 자동 재시작
womp 1 켬 Wake on LAN (네트워크로 깨우기)

[2] OpenClaw Gateway 자동 실행

부팅 시 Gateway가 자동으로 시작되도록 설정합니다.

Terminal 
# launchd 서비스 설치
openclaw daemon install

# 서비스 상태 확인
openclaw daemon status

# 서비스 시작
openclaw daemon start

[3] 설정 확인

Terminal 
# 전원 관리 설정 확인
pmset -g

# OpenClaw 상태 확인
openclaw status

예상 출력

pmset -g 결과 
System-wide power settings:
 sleep             0
 disksleep         0
 displaysleep      0
 hibernatemode     0
 standby           0
 autopoweroff      0
 autorestart       1
 womp              1

[RESULT] 결과

이 설정 후:

  • Mac Mini가 절대 잠들지 않음
  • 정전되어도 전원 복구 시 자동으로 켜짐
  • OpenClaw Gateway가 부팅 시 자동 실행됨
  • Wake on LAN으로 원격에서 깨울 수 있음
[i] 추가 팁

Mac Mini를 headless(모니터 없이)로 운영하려면 HDMI 더미 플러그나 시스템 환경설정에서 "화면 공유" 활성화를 권장합니다.
