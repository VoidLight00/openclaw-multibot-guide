OpenClaw 멀티봇 가이드

하나의 Mac Mini에서 여러 AI 에이전트를 운영하는 완벽 가이드.
Claude, Gemini, GPT를 각각의 텔레그램 봇으로 분리하고 관리하세요.

OpenClaw 2026.2.1+ macOS Telegram

[TARGET] 이 가이드에서 배우는 것

[BOT] 멀티 에이전트

하나의 OpenClaw에서 여러 에이전트를 동시에 운영하는 방법

[MOBILE] 텔레그램 멀티봇

각 에이전트를 독립적인 텔레그램 봇으로 연결하는 방법

[AI] AI 모델 설정

Claude, Gemini, GPT 등 다양한 AI 모델 연동 및 변경

[CONFIG] Config 마스터

agents, bindings, workspace 등 핵심 설정 완벽 이해

[QUICK] 빠른 시작

[i] 이미 OpenClaw가 설치되어 있나요?

설치가 완료된 상태라면 5. Config 설정으로 바로 이동하세요.

OpenClaw 설치

npm으로 간단하게 설치합니다.

Terminal 
npm install -g openclaw

초기 설정

대화형 마법사로 기본 설정을 완료합니다.

Terminal 
openclaw setup

멀티봇 Config 적용

이 가이드를 따라 멀티 에이전트 설정을 완료합니다.

Gateway 실행

모든 봇이 동시에 작동합니다!

Terminal 
openclaw gateway start

[DOCS] 목차

장 내용 소요 시간
1. 개요 멀티 에이전트가 필요한 이유와 핵심 개념 5분
2. 사전 준비 OpenClaw 설치, 텔레그램 봇 생성 15분
3. AI 모델 설정 Claude, Gemini, GPT OAuth/API 연동 10분
4. Workspace 구조 에이전트별 작업 공간 설정 10분
5. Config 설정 핵심! agents, bindings, channels 완벽 이해 20분
6. 실행 & 테스트 Gateway 시작, 봇 테스트 5분
[!] 주의사항

같은 텔레그램 봇 토큰을 여러 기기에서 동시에 사용하면 충돌이 발생합니다. 멀티봇 설정 전에 다른 기기의 OpenClaw를 중지하세요.
